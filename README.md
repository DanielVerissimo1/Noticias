**Portal de Notícias** é um desafio de responsividade desenvolvido durante a formação Full-Stack da **Rocketseat**. O objetivo é levar a homepage de um portal de notícias sobre tecnologia para o próximo nível, tornando-a completamente responsiva e adaptável a diferentes tamanhos de tela.

---

## 📸 Visualização do Projeto:

<img alt="preview" src="/Thumbnail (3).png" />

---

## 🚀 Funcionalidades:

- Homepage de portal de notícias sobre tecnologia
- Layout responsivo para diferentes dispositivos (mobile, tablet, desktop)
- Organização de cards de notícias
- Menu de navegação adaptável
- Estrutura semântica com HTML
- Design moderno e focado em leitura

---

## 🛠️ Tecnologias utilizadas:

- HTML5  
- CSS3 (com foco em media queries e responsividade)
